# Plan-Server: BuySell Web Application

A lightweight and fast Full-stack web application built to simulate a dynamic Buy & Sell marketplace. This project demonstrates backend routing, server-side rendering, and seamless synchronous file system database management using Node.js and Express.

---

## 🚀 Features & Project Overview

* **Server-Side Rendering:** Utilizing EJS templates to deliver clean, dynamic HTML pages directly from the server.
* **Non-Blocking Architecture:** Built on top of Node.js event-driven, asynchronous environment ensuring reliable scalability.
* **Mock Database Integration:** Implements streamlined JSON file-based database routines using `fs.readFileSync` for instant data availability upon server startup.

---

## 🛠️ Tech Stack & Architecture

* **Backend Environment:** Node.js
* **Web Framework:** Express.js
* **Template Engine:** EJS (Embedded JavaScript)
* **Database / Storage:** JSON-based local storage (`database/userdata.json`)
